Deskripsi

YourFav Florist adalah website toko bunga handmade yang dirancang untuk memudahkan pemilik toko dalam mengelola produk bunga. Website ini menampilkan koleksi buket bunga dengan berbagai kategori dan memungkinkan admin untuk menambahkan produk baru.

Fitur
Navbar - Menu navigasi dengan link ke Beranda, Produk, dan Tambah Bunga
Hero Section - Bagian beranda 
Statistik - Menampilkan 3 kartu statistik (Total Produk, Stok Tersedia, Kategori)
Produk - Menampilkan 3 produk bunga dengan foto, harga, dan stok
Form - Form untuk admin menambahkan produk baru dengan fields:
- Nama Buket
- Harga (Rp)
- Stok
- Kategori
Footer - Bagian bawah dengan informasi toko
